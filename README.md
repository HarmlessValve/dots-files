# Dots Files

My personal dotfiles configuration.

## Structure

```
dots-files/
├── backgrounds/          # Wallpaper images
├── fastfetch/            # Fastfetch configuration
├── ghostty/              # Ghostty terminal config + shaders
├── hypr/                 # Hyprland WM configuration
├── screensaver/          # Screensaver config
├── shell/                # Setup and pull scripts
├── starship/             # Starship prompt config
├── walker-styles/        # Walker theme styles
├── waybar/               # Waybar status bar config
└── zsh-conf/             # ZSH configuration
```

## Installation

### Setup (deploy dotfiles to system)

```bash
./setup.sh
```

### Pull (backup system configs to dots)

```bash
./pull.sh
```
